# 📱 Telecom Customer Churn Prediction

![Python](https://img.shields.io/badge/Python-3.8+-blue.svg)
![ML](https://img.shields.io/badge/ML-Churn_Prediction-blue.svg)

## 📖 Overview

Predictive model to identify customers likely to churn from telecom services. Analyzes customer demographics, usage patterns, and service data to predict churn probability.

## 🎯 Objectives

- Predict customer churn probability
- Identify key churn factors
- Provide retention recommendations
- Customer segmentation analysis

## 🛠️ Tech Stack

- Scikit-learn, XGBoost
- Pandas, NumPy
- Matplotlib, Seaborn

## 🚀 Usage

```bash
git clone https://github.com/adityavdn/Telecom-Customer-Churn-Prediction.git
pip install -r requirements.txt
jupyter notebook
```

## 📊 Key Insights

- Top churn indicators
- Customer lifetime value analysis
- Churn rate by demographics
- Retention strategy recommendations
